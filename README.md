# 🔊 Text to Voice Converter

A simple and interactive **Text to Voice Converter** built using **HTML, CSS, and JavaScript**. This web application converts user-entered text into speech using the browser's built-in Speech Synthesis API.

## 📌 Features

- 📝 Enter any text and convert it into speech.
- 🌍 Supports multiple languages and voices (based on browser support).
- 🎙️ Select different available voices.
- 🎨 Clean and responsive user interface.
- ⚡ Lightweight and runs directly in the browser.

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- Web Speech API (SpeechSynthesis)

## 🚀 Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/Shubo89/text-to-voice-converter.git
   ```

2. Open the project folder.

3. Double-click **text to voice.html** or open it using Live Server in VS Code.

4. Type any text, select a voice, and click the speak button.

## 💡 How It Works

- Users enter text into the text area.
- JavaScript retrieves the available system voices.
- The selected voice is applied using the Speech Synthesis API.
- The browser reads the text aloud.
